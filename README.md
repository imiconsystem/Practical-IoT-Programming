# ชุดเรียนรู้การโปรแกรม IoT (Practical IoT Programming)
เรียนรู้การโปรแกรม IoT เชิงปฏิบัติด้วย ESP32 NodeMCU พร้อมอุปกรณ์และคู่มือใบงานแนว “Keep It Simple” เรียบง่าย ใช้เวลาเรียนน้อย เข้าถึงรวดเร็ว
